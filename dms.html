Delivery Management System

Feb. 2016

Back to Homepage

A system that I helped develop for La Belle Thailandaise to contact drivers for deliveries.

  • Nexmo SMS PHP SDK to send and receive SMS
  • Bootstrap to layout pages
  • JavaScript, jQuery and AngularJS for client side processing
  • JavaScript Google Maps API for delivery location and cost
  • JSON and AJAX to communicate with backend process
  • PHP for backend processing
  • PHP fpdf library to generate delivery bills
  • MySQL for database

To contact a delivery driver, you input the customer's address into the Address field - you will get suggestions as you type it out. Once the address has been inputted, it will display the distance from the restaurant as well as the driving fee to pay to the delivery driver.

From there, you select a delivery person to contact and press the appropriate button. This will send a text message to the selected driver and then the application will wait for a reply within the next 5 minutes. If a reply is received, the driver will be officially associated with the delivery and should come to the restaurant to pickup the food.

Nice features of this application include things like the text message system - with a panel on the right displaying all incoming and outgoing messages, automatic calculation of driver fees to be paid and every request is saved in a database and can be accessed at anytime and anywhere.
