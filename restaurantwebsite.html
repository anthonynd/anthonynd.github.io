In the month of March 2016, I redesigned and developed a responsive restaurant website for La Belle Thailandaise.

  • HTML/CSS for web page structure
  • Bootstrap for some styling, layout and responsiveness
  • Javascript/jQuery for animations and event handling
  • PHP to connect to a MySQL database needed for menu display and online reservations
  • AngularJS as a front-end framework to display language-specific text strings
  • Vegas for top-page carousel
  • Google Maps API to display restaurant location and provide directions

The site features a carousel at the top of the page showcasing relevant photographs of the restaurant. On desktop displays, there are 4 navigational buttons as well as social media buttons floating on top of the carousel. Mobile users can navigation through a fixed collapsible menu that is opened by tapping on the button in the top-right corner.

The menu showcases a parallax background where menu items scroll by as the user scrolls down the page. Below that, users can make an online reservation where they must properly provide date, time, name etc. At the bottom, users can see contact information and the location of the restaurant on Google Maps where they can quickly find directions.
